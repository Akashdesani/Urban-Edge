# **App Name**: Urban Edge

## Core Features:

- Product Catalog: Display a curated catalog of T-shirts, fetched dynamically from Firestore.
- Dynamic Product Pages: Create individual pages for each T-shirt with detailed information and customer reviews.
- Interactive Shopping Cart: Enable users to add, remove, and adjust quantities of items in their cart, persisted via Firebase.
- Secure Checkout Process: Provide a seamless checkout experience, including delivery details, order summary, and payment options.
- User Authentication: Implement Firebase Authentication for user registration, login, and access to protected pages.
- Admin Product Management: Allow admins to add, edit, and delete products, as well as view and update orders.
- AI-Powered Style Assistant: Offer an AI-powered tool that allows users to upload a photo of their outfit and receive style recommendations for matching T-shirts.

## Style Guidelines:

- Primary color: Black (#0f0f0f) to establish a dark, bold foundation.
- Background color: Dark grey (#222222) providing depth while maintaining the dark theme, 20% saturation.
- Accent color: Electric blue (#7DF9FF) used sparingly to highlight key interactive elements, creates contrast.
- Headline font: 'Space Grotesk', sans-serif, for a modern and bold title.
- Body font: 'Inter', sans-serif, for clear and accessible descriptions. Note: currently only Google Fonts are supported.
- Minimalistic icons to complement the modern design, with neon/electric blue accents on hover.
- Mobile-first, responsive design using a grid layout for optimal viewing across devices, incorporating glassmorphism effects for a premium look.
- Subtle hover glow effects and micro-animations for a smooth and engaging user experience.
